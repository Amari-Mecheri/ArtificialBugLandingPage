# ArtificialBug Landing Page

Welcome to the ArtificialBug Landing Page repository! This project contains the source code for the landing page of ArtificialBug, a platform dedicated to innovative solutions in the tech industry.

## About
ArtificialBug is a cutting-edge platform designed to revolutionize the way we approach technology and problem-solving. Our landing page serves as the gateway to our services, providing visitors with an overview of what we offer and how we can help them achieve their goals.

## Live Website
The landing page is hosted at: [https://artificialbug.com](https://artificialbug.com)

## Features
- Modern and responsive design
- Clear and concise information about ArtificialBug
- Easy navigation for users

## Functionalities

The ArtificialBug landing page includes the following features:

### Navigation Bar
- Fixed at the top with links to sections like "Features," "Categories," and "Early Access."
- Includes a "Get Started" button that focuses on the email input field.

### Hero Section
- A visually appealing introduction with a gradient text title, tagline, and call-to-action buttons ("Join Waitlist" and "Learn More").

### Features Section
- Highlights the platform's benefits with cards for:
  - Curated Quality
  - Deploy in Seconds
  - Fair Pricing
  - Customizable
  - Analytics Dashboard
  - Creator Economy

### Categories Section
- Displays various AI agent categories using cards, including:
  - Customer Support
  - Development
  - Data Analysis
  - Content Creation
  - Marketing
  - Research
  - Design
  - Productivity

### Early Access Section
- Includes a form for users to join the waitlist by submitting their email.
- The form is connected to an Azure Function for backend processing.

### Footer
- Contains links to sections and a contact email.
- Credits the product to "UptechLLC."

### JavaScript Functionalities
- Smooth scrolling for navigation links.
- Form submission logic with error handling and feedback messages.
- Integration with an Azure Function for email submissions.
